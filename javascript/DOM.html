문서 객체 모델 (DOM, Document Object Model)

  • DOM트리

    • 웹 문서에 있는 요소들 간의 부모, 자식 관계를 계층 구조로 표시한 것
    • JavaScript가 이해 할 수 있도록 객체화 한 모델 구조 => DOM
    • 나무 형태로 표현이 되기 때문에 'DOM 트리'라고 함.
    • node : DOM 트리에서 가지가 갈라져 나간 항목
    • root node : DOM 트리의 시작 부분 => html
    • 모든 HTML 태그는 element(요소) node
    • 웹 문서의 텍스트 내용은 요소 node의 자식 node인 text node
    • 태그의 속성 : element node의 자식 node인 attribute(속성)node
    • 주석 : 주석(comment) node

  • 노드의 타입

    • 문서 노드 (Node.DOCUMENT_NODE) : 최상위 DOCUMENT의 객체의 node type
    • 요소 노드 (Node.ELEMENT_NODE) : 요소의 node type => h1, p tag ,,,,
    • 속성 노드 (Node.ATTRIBUTE-NODE) : 속성의 node type => href , src ,,,,
    • 텍스트 노드 (Node.TEXT_NODE) : 텍스트에 해당하는 node type
    • 주석 노드 (Node.COMMENT_NODE) : 주석에 해당하는 node type

  • DOM 요소에 접근 방법 documet.

    • getElementById() 메소드
      • 형식 : 요소명.getElementById("id 이름")
    • getElementByClassName() 메소드
      • 형식 : 요소명.getElementByClassName("class이름")
    • getElementByName() 메소드
      • 형식 : 요소명.getElementByName("tag명")
    • querySelector() 메소드
      • 형식 : 요소명.querySelector("선택자(id,class)")
      • 형식 : 요소명.querySelector("#id명")
      • 형식 : 요소명.querySelector(".class명")
      • 형식 : 요소명.querySelector("태그명")
    • querySelectorAll() 메소드
      • 형식 : 요소명.querySelectorAll("선택자or태그")

  • 노드 추가하기~

    • createElement(태그명)
    • 형식 : document.createElement("a")
  • 요소노드에 텍스트 추가하기~
    • document.createTextNode("입력한 문자")
    • 부모노드.appendChild(텍스트노드)

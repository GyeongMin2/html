브라우저 객체 모델(Browser Object Model)의 종류

  • window : 웹 브라우저가 열릴 때 마다 생성되는 최상위 객체
    • innerWidth : 웹 브라우저의 화면 너비
    • innerHeight : 웹 브라우저의 화면 높이
    • outerWidth : 웹 브라우저 창의 너비
    • screenTop/screenY : 웹 브라우저 위쪽 면과 모니터의 간격
      • ]
    • screenLeft/screenX : 웹 브라우저 왼쪽면과 모니터의 간격
    • pageXOffset/scrollX : 웹 브라우저의 수평 스크롤 위치
    • pageYOffset/scrollY : 웹 브라우저의 수직 스크롤 위치

      주요메서드

    • alert() : 알림창 표시
    • confirm() : 확인창 표시
    • prompt() : 입력창 표시
    • open() : 새로운 웹 브라우저 창 열기
    • close() : 웹 브라우저 닫기
    • setTimeout() : 일정시간 (ms) 뒤에 콜백함수를 한번만 실행
    • setInterval() : setInterval() 메소드로 반복 실행되는 함수를 중지
    • scrollto() : 웹 브라우저의 스크롤을 특정 위치로 이동
    • scrollBy() : 웹 브라우저의 스크롤을 현재 위치에서 상대위치로 이동
    • document : 웹 브라우저에 표시되는 HTML 문서 정보가 포함된 객체 -> <html> </html>
    • location : 웹 브라우저에 현재 포함된 페이지에 대한 URL 정보가 포함된 객체
    • history : 웹 브라우저에 저장된 방문 기록이 포함된 객체
    • navigator : 웹 브라우저(메뉴) 정보가 포함된 객체
    • screen : 웹 브라우저의 화면 정보가 포함된 객체

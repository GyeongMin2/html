이벤트

  • 웹 브라우저나 사용자가 행하는 동작
  • 웹 문서 (DOM : document object model) 영역 안에서 발생하는 동작으로 한정
  • 마우스, 키보드, 웹문서 로딩시, 폼 내용 작성 등...

이벤트의 종류

    마우스 관련 이벤트

    on ~ :
  • click : html 요소 || 영역을 클릭할때 발생
  • dblclick : 두 번 클릭할 때 발생
  • onmouseover :
  • onmouseout :
  • onmousemove :
  • onmousedown :
  • onmouseup :
  • onwheel :

    키보드 관련 이벤트

  • onkeypress : 키를 누를때
  • onkeydown : 키를 누르는 동안 발생
  • onkeyup : 누르던 키가 올라올때 발생, (손을 땔 때)

    포커스 이벤트

  • onfocus : 요소에 포커스(커서,포인터)
  • onblur : 요소가 포커스를 잃으면 발생

    폼이벤트

  • onsubmit : 폼 전송 될때
  • reset : 폼 요소 값의 초기화

    문서 관련이벤트

  • onload : 문서 로딩이 끝나면
  • resize : 문서 화면의 크기가 바뀌면
  • scroll : 화면이 스크롤되면 발생
  • abort : 문서가 완전히 로딩되기 전에 멈추면 발생
  • error : 문서가 정확히 로딩되지 않고 에러가 발생할 경우
  • unload : 문서에서 벗어날때 발생

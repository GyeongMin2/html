함수 function

  • 어떤 목적을 가지고 작성한 코드를 모아놓은 블록문
  • 각 명령의 시작과 끝을 명확히 구분 할 수 있음
  • 함수를 정의한다 :
      블록문을 구성한다.
    • function 키워드
    • 식별자
    • 소괄호( )
    • 등이 필요
  • 코드의 재활용이 가능하다.
  • function 함수명(매개변수){ 실행구문; 함수; 로직; 변수명; etc...; [return 값;] }
  • 함수 표현식
    • 익명함수
      • let 변수명 = function (매개변수,,){}
      • 변수명(매개변수,,)
      • 변수명 ( 10,20 )
      • function sum(매개1,매개2){} sum(10,20)
    • 네이밍 함수 생성자 함수
      • 변수에 할당하는 함수에 식별자가 있을 때
      • const 변수명 = function 이름(){};
    • 즉시 실행 함수
      • (function(매개변수,,,){실행구문;}());
      • (function(매개변수,,){실행구문;,,,}(인수 : 이벤트값, li.1,li.2 ));
    • 화살표 함수
      • ES6에서 정의
      • ( )=>{ }
      • const sum = ( )=>{로직};

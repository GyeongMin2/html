• 플렉스 박스 레이아웃
    • 수평방향, 수직방향 중 하나를 축으로 정하고 박스를 배치
    • 여유 공간이 생길경우, 너비나 높이를 적절하게 늘리거나 줄일 수 있음
    • 주축 : 플렉스 박스의 진행 방향
    • 교차축 : 주축과 수직방향
    • 플렉스 컨테이너 : display 속성 값으로, flex, inline-flex 사용
    • 플렉스 아이템 : 플렉스 컨테이너의 자식관계를 이루는 요소
  • 플렉스 박스의 기본 속성
    • display 속성
      • flex → 컨테이너 안의 플렉스 항목을 블록레벨 요소로 배치
      • inline-flex → 플렉스 항목을 인라인 레벨 요소로 배치
      • 형식 → display:flex | inline-flex
    • flex-direction
      • 플렉스 박스 레이아웃의 주축 방향 지정
      • 형식 → flex-direction : 속성값
      • 속성값 → low : 주축 방향을 왼쪽→오른쪽
      • 속성값 → row-reverse : 주축 방향을 오른쪽→왼쪽
      • 속성값 → column : 주축 방향을 위 → 아래
      • 속성값 → column-reverse : 주축 방향을 아래 → 위
    • flex-wrap 속성
      • 플렉스 아이템의 자동 줄바꿈 여부 처리
      • 형식 → flex-wrap : 속성값
      • 속성값 → nowrap : 아이템이 플렉스 컨테이너의 영역을 벗어나도 무시
      • 속성값 → wrap : 아이템이 컨테이너 영역을 벗어나면 줄바꿈
      • 속성값 → wrap-reverse 아이템이 컨테이너 영역을 벗어나면 wrap의 역방향으로 줄바꿈
    • flex-flow 속성
      • flex-direction 과 flex-wrap 속성을 한번에 적용할 수 있는 단축속성
      • 형식 → flex-flow: direction wrap
      • 기본값 → flex-flow: low no-wrap
    • justify-content 속성
      • 플렉스 아이템을 모두 주축 방향으로 정렬
      • 형식 → justify-content : 속성값
      • 속성값 → flex-start : 주축 방향의 시작점 기준으로 정렬
      • 속성값 → flex-end : 주축 방향의 끝점 기준으로 정렬
      • 속성값 → center : 주축 방향의 중앙을 기준으로 정렬
      • 속성값 → space-between : 플렉스 아이템 사이의 간격이 일정하도록 정렬
      • 속성값 → space-around : 아이템의 둘레가 일정하도록 정렬
      • 속성값 → space-evenly : 아이템의 사이, 양쪽 끝 간격이 일정하도록 정렬
    • align-items,align-self 속성
      • 플렉스 아이템을 모두 교차축방향으로 정렬
      • 교차축에서 특정 항목만 지정할 경우 align-self
      • 형식 align-items : 속성값
      • 속성값 → flex-start : 교차축 방향의 시작점을 기준으로 정렬
      • 속성값 → flex-end : 교차축 방향의 끝점을 기준으로 정렬
      • 속성값 → center : 교차축 방향의 중앙을 기준으로 정렬
      • 속성값 → baseline : 플렉스 아이템의 baseline 기준으로 정렬
      • 속성값 → stretch : 교차축 방향으로 플렉스 아이템의 너비,높이가 늘어남
    • align-content 속성
      • 플렉스 아이템이 두 줄 이상일때 교차축 방향으로 정렬
      • 형식 → align-content : 속성값
      • 속성값 → flex-start : 교차축 시작점에서 맞춰 배치
      • 속성값 → flex-end : 교차축 끝점에 맞춰 배치
      • 속성값 → center : 교차축 중앙에 맞춰 배치
      • 속성값 → space-between : 첫 항목, 끝 항목은 교차축의 시작과 끝에 맞추고, 사이 간격은 같은간격으로 배치
      • 속성값 → space-around : 교차축에 모든 항목을 같은 간격으로 배치
      • 속성값 → stretch : 교차축 방향으로 플렉스 아이템의 너비,높이

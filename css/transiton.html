트랜지션

  1. 웹 요소의 스타일 속서을 시간차를 두고 조금씩 바뀌게 설정하는 것
  2. 어떤 요소의 속성값을 다른 값으로 변경하는 것
  3. transition-property 속성
    • 트랜지션의 대상을 지정
    • 전화 효과의 해당 속성명을 값을 지정
    • 형식 : transition-property:<속성값>
    • none : 전환 효과 속성을 지정 안함
    • all : 모든 속성을 전환 효과 대상으로 지정
    • 속성 이름 : 트랜지션 효과를 적용할 속성이름, 쉼표(,)로 구분
  4. transition-duration 속성값
    • 트랜지션 진행시간 설정
    • 어떤 요소에 전환 효과를 적용시 반드시 property, duration을 함께 사용
    • 여러개 일 경우, 쉼표(,)로 구분하여 적용
  5. transition-delay 속성값
    • 전환 효과가 지연되는 시간을 지정
    • 형식 : transition-delay:<시간>
  6. transition-timing-function 속성
    • 트랜지션의 선환 효과 속도를 지정
    • 트랜지션의 시작, 중간, 끝 지점의 속도를 지정
    • 형식 : transition-timing-function:<속성값>
    • linear : 처음, 마지막 속도 일정
    • ease : 처음에는 빨리 진행하다가 중간지점부터 점점 느려짐
    • ease-in : 처음에는 느리다가 점점 빨라짐
    • ease-out : 처음에는 빠르다가 점점 느려짐
    • ease-in-out : 처음에는 느리다가 점점 빨라지다가 다시 느려짐
    • cubic-bezier(n,n,n,n) 베지에 함수를 이용하여 정의, n값은 0~
  7. transition 속성
    • 위의 속성값을 한꺼번에 지정 가능
    • 형식 : transition:<property>, <duration>, <timing-function>, <delay>
    • 시간속성을 앞에것은 duration

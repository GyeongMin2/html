변형(transform)

  1. 변형(transform) : 특정 요소의 크기나 형태, 스타일 등이 바뀌는것.
    • 2차원 변형
      • 수평/ 수직으로 웹 요소를 변형
      • 크기/각도만 지정
      • 2차원 좌표 사용
      • 확대/ 축소
        • scale(x, y) → 요소를 x,y만큼 확대/축소
        • scaleX(n) → 요소를 n만큼 x축으로 확대/축소
        • scaleY(n) → 요소를 n만큼 y축으로 확대/축소
        • scale3d(x, y, z) → 요소를 x,y,z만큼 확대/축소
      • 기울이기
        • skew(xdeg, ydeg) → 요소를 x축, y축으로 xdeg, ydeg만큼 기울임
        • skewX(xdeg) → 요소를 x축으로 xdeg만큼 기울임
        • skewY(ydeg) → 요소를 y축으로 ydeg만큼 기울임
      • 회전
        • rotate(deg) → 요소를 deg만큼 회전
      • translate 함수 → 지정한 방향으로 이동할 거리 지정 후 이동
        • transform:translate(tx,ty) → x축 방향으로 tx만큼, y축 방향으로 ty만큼 이동 ,ty 값을 주지 않으면 0으로 간주.
        • transform:translate3d(tx,ty,tz) → x축 방향으로 tx만큼, y축 방향으로 ty만큼 이동 ,ty 값을 주지 않으면 0으로 간주.
        • translate(tx) → x축방향으로
        • translate(ty) → y축방향으로
        • translate(tz) → z축방향으로
